# SOC seed · sector C · Industria Manufacturera

> Concepto invariante que aplica a cualquier proyecto SOS dentro del
> sector CNAE C. La IA usa este SOC como contexto al generar el
> mapa de valor de proyectos de este sector.

## Esencia
transformación de materias primas · cadena de valor estandarizada · escala.

## Adaptación recomendada de roles
- Usar nomenclatura oficial CNAE C cuando se conozca el sub-sector.
- Roles canónicos (founder · operations · creator · reviewer · facilitator)
  se traducen a vocabulario del sector (ej. para Industria Manufacturera · adaptar al
  vocabulario profesional habitual).

## SOPs derivables (futuro)
- Procesos operativos canónicos del sector quedan pendientes de
  identificar conforme aparezcan recurrencias entre clientes.
